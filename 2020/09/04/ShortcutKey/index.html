IDEA快捷键大全

  • Tool
  • 2020-09-04

Crtl

快捷键 介绍
Crtl+F 在当前文件内进行文本查找
Crtl+R 在当前文件内进行文本替换
Crtl+C 复制光标所在行或复制选择内容
Crtl+X 剪切光标所在行或复制选择内容
Crtl+V 粘贴复制或剪切的内容
Crtl+Y 删除光标所在行或删除选中的行
Crtl+D 复制光标所在行，并粘贴内容至光标下行；复制选择内容并粘贴至光标后方
Crtl+Z 撤销
Crtl+W 递进式选择代码块。可选中光标所在的单词或段落，连续按会在原有选中的基础上再扩展选中范围
Crtl+E 显示最近打开的文件记录列表
Crtl+N 根据输入的类名查找类文件
Crtl+G 在当前文件跳到指定行
Crtl+J 插入自定义动态代码模板
Crtl+P 方法参数提示显示
Crtl+Q 光标所在的变量 / 类名 / 方法名等上面（也可以在提示补充的时候按），显示文档内容
Crtl+B 进入光标所在的方法/变量的接口或是定义处，等效于 Ctrl + 左键单击
Crtl+H 显示当前类的层次结构
Crtl+O 选择可以重写的方法
Crtl+I 选择可以继承的方法
Crtl++ 展开代码
Crtl+- 折叠代码
Crtl+/ 注释光标所在行代码，会根据当前不同文件类型使用不同的注释符号
Crtl+[ 移动光标到当前所在代码的花括号开始位置
Crtl+] 移动光标到当前所在代码的花括号结束位置
Crtl+F1 在光标所在的错误代码处显示错误信息
Crtl+F3 跳转到所选中词的下一个引用位置
Crtl+F4 关闭当前编辑文件
Crtl+F8 在 Debug 模式下，设置光标当前行为断点，如果当前已经是断点则去掉断点
Crtl+F9 重新编译当前文件
Crtl+F11 选中文件 / 文件夹，使用助记符设定 / 取消书签
Crtl+F12 弹出当前文件结构层
Crtl+Tab 编辑窗口切换，如果在切换的过程又加按上Delete，则是关闭对应选中的窗口
Crtl+End 跳到文件尾
Crtl+Home 跳到文件头
Crtl+Delete 删除光标后的单词或中文句
Crtl+BackSpace 删除光标前的单词或中文句
Crtl+0…9（A..Z） 定位到对应数值的书签位置（不能使用小键盘的数字按键）
Crtl+左键单击 在打开的文件标题上，弹出该文件路径
Crtl+左方向键 光标跳转到当前单词 / 中文句的左侧开头位置
Crtl+右方向键 光标跳转到当前单词 / 中文句的右侧开头位置
Crtl+前方向键 等效于鼠标滚轮向前效果
Crtl+后方向键 等效于鼠标滚轮向后效果
Crtl+K 版本控制提交项目，需要此项目有加入到版本控制才可用
Crtl+T 版本控制更新项目，需要此项目有加入到版本控制才可用
Crtl+U 前往当前光标所在的方法的父类的方法 / 接口定义

Shift

快捷键 介绍
Shift+F1 如果有外部文档可以连接外部文档
Shift+F2 跳转到上一个高亮错误 或 警告位置
Shift+F3 在查找模式下，查找匹配上一个
Shift+F4 对当前打开的文件，使用新Windows窗口打开，旧窗口保留
Shift+F6 对文件 / 文件夹 重命名
Shift+F7 在 Debug 模式下，智能步入。断点所在行上有多个方法调用，会弹出进入哪个方法
Shift+F8 在 Debug 模式下，跳出，表现出来的效果跟 F9 一样
Shift+F9 等效于点击工具栏的 Debug 按钮
Shift+F10 等效于点击工具栏的 Run 按钮
Shift+F11 弹出书签显示层
Shift+Tab 取消缩进
Shift+End 选中光标到当前行尾位置
Shift+Home 选中光标到当前行首位置
Shift+Enter 开始新一行。光标所在行下空出一行，光标定位到新行位置
Shift+左键单击 在打开的文件名上按此快捷键，可以关闭当前打开文件
Shift+滚轮前后滚动 当前文件的横向滚动轴滚动
